## Slide 3 — 페인포인트: 금융 소외

**헤드라인:** "제도권은 외면, 사채는 착취"

**한 줄 메시지:** 한국 금융 시스템은 이들을 보호하지 못합니다. 마중이 그 공백을 채웁니다.

## Slide 6 — 아키텍처: LLM은 실행 경로에 없습니다

**헤드라인:** "가장 중요한 순간에 LLM은 완전히 배제됩니다"

```
고객 발화(자연어)
    ↓ [LLM: 의도 파싱만]
Gate A — 위임장 검증 (Mandate Validation)
    ↓ PASS
Gate B — Rule 한도·조건 (환율·금액·월한도)
    ↓ PASS
Gate C — 수취인 화이트리스트 + AML 스크리닝 (점수 40/70 컷)
    ↓ PASS → 송금 실행 / HOLD → STR 대기열 / BLOCK → 즉시 차단
```

**한 줄 메시지:** 환각으로 인한 사고를 원천 차단합니다. 수집 → 판단 → 행동 → 검증, 4단계 안전 설계.

## Slide 9 — 규제 대응

**헤드라인:** "특금법 준수 설계 — 무증빙 한도 내 안전 운영"

- **STR·CDD·CTR 의무** 준수 *(KoFIU 특금법)*
- 외국인 거주자 무증빙 송금 연 5만 달러 한도 이내 운영 *(은행연합회)*
