## 규제 3줄 요약 (심사위원 즉답용)

1. **접근매체 비양도**: 마중은 인증수단을 받지 않음. 오픈뱅킹 자동이체 권한 위에서 JB 코드가 실행. 법원 판례(2011도16167)상 양도 요건 미해당.
2. **LLM 차단**: LLM은 `ParsedIntent`(후보값)만 반환. `execute_remittance`를 아키텍처상 호출 못 함. 3중 게이트가 서명된 위임장으로 재검증.
3. **망분리 정합**: LLM에 흐르는 것은 자연어 발화뿐. 계좌·실명·금액은 결정적 코드 내부. 금융위 2024.8 망분리 로드맵 허용 범위.
