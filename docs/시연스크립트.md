# 마중(Majung) 시연 영상 스크립트 (5분, 예선 제출용)

> 원칙: 베트남어 대사는 아래 고정 10문장만 사용하고, 화면 안에 한국어 병기 자막을 필수로 노출한다.
> 시나리오 ②는 고객 개입 없는 자동 실행 장면을 반드시 포함한다.
> 촬영 화면 구성: 고객 폰 프레임(좌)과 데모 컨트롤 패널, 관리자 대시보드는 별도 전체 화면 컷으로 촬영.

## 라이브 시연 런북 (본선 7/4~5 현장용, 8분)

### 시연 전 점검 (D-30분)
1. `cd backend && python -m app.seed` — DB 완전 초기화
2. `python -m uvicorn app.main:app --port 8000` — 백엔드 기동, `curl localhost:8000/health` 확인
3. `cd frontend && npm run dev` — 프론트 기동, 브라우저 5173 접속
4. **상단 "API 연결됨" 녹색 표시** 반드시 확인 (없으면 오프라인 목 모드로 전환됨)
5. 브라우저 배율 100%, 창 폭 1280px 이상, 화면 전체화면 준비
6. 폰 프레임이 화면 정중앙에 오는지 확인. 우측 패널은 **F키로 숨김** 후 데모 진행

## 촬영 환경 셋업 (데모 단축키)

- 기동 순서: 백엔드 폴더에서 `python -m app.seed` → `python -m uvicorn app.main:app --port 8000`, 프론트 폴더에서 `npm run dev` → 브라우저 5173 접속. 상단 "API 연결됨" 녹색 표시 확인.
- 촬영 창 폭 1180px 이상, 브라우저 배율 100%. 넓은 화면에서는 컨트롤 패널이 우측 고정되어 폰 프레임이 화면 정중앙에 온다.
- **F 키**: 컨트롤 패널 숨김/표시. 패널을 숨겨도 단축키는 동작하므로 화면에 폰만 두고 진행 가능(② 자동 실행 컷에서 커서·패널 미노출에 유리).
- **숫자 1~5 키**: 데모 ①~⑤ 단계 실행. 채팅 입력창에 포커스가 있으면 단축키는 무시되어 베트남어 타이핑과 충돌하지 않음.
- 연출 지연 내장: 급여 감지 후 약 0.9초 뒤 자동 송금 카드가 등장하고 ② 카드에 녹색 글로우가 1.6초간 표시되어 "감지 → 판정 → 실행" 순서가 화면에서 읽힌다. 버튼 연타·동시 클릭은 자동 차단(1회만 실행).
- **재테이크 주의**: 한 테이크에서 ②를 두 번 누르면 두 번째는 월 한도 소진으로 "거절" 카드가 나온다(게이트의 정상 판정). 재촬영 전 반드시 `python -m app.seed` 재실행 후 화면 하단 "피드 초기화" 클릭.
