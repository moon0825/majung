## 신설 슬라이드 1 — 대환 메커니즘 (여신·컴플라이언스 최대 급소 해소) `[일부 B]`
> 핵심 교정: "대환 = 사채를 갚아준다"(오해) → **"신용을 만들어 제도권으로 갈아타되, 돈이 실제 사채를 끄는지까지 검증한다."**

```
④ [연 30% 사채]  ──상환──▶  소멸
        ▲  자가청산 검증: 사채계좌 직접 상환 / 수취계좌 '등록 대부업자' 대조
        │  → 미검증·불법사채 계좌면 송금 대신 보류 + STR  (자금세탁 통로 차단) [B]
③ [JB 외국인 신용대출 연 13.59%]   ※ 실행·최종승인 = JB 심사엔진 (가심사≠승인)
        ▲
② [가심사 적격 게이트]  비자 잔여 > 만기 + 6개월 완충 · DSR ≤ 0.40 · 다중채무 < 3 · 급여 연속성
        │  + 사업장 이탈·계약해지 이벤트 감지 시 한도동결/조기상환  ← 출국·회수 리스크 [B]
        ▲
① [1단계 자동송금이 3개월간 급여·거래 데이터 적립 → 내부등급 X→Y 상승]  ◀ 핵심 인과
        ▲
  [신규 도착자: 소득이력 0 → 제도권 거절 → 사채로 유입]  (신한 '3개월 급여' 공백)
```
- 멘트: "직접 갚아주는 게 아닙니다. **송금이 쌓은 데이터로 자격을 만들고**, JB 대출로 갈아타되, **그 돈이 실제 사채를 끄는지까지 코드가 봅니다.** 불법 사채 계좌면 송금 대신 STR입니다."
- 이 한 장이 여신("자금사용 검증 없다")·컴플라이언스("불법조직 재유입") 두 급소를 동시에 닫는다.

## 신설 슬라이드 2 — 숫자: 단일 큰 수 → 위험조정 워터폴 (여신·사업 급소)
> "66.5억"을 버리고 **시나리오 범위 + 위험조정**으로. 분모는 **검증된 24만**만(휴면 20만은 분모 제외·옵션 격리).

깔때기(잔존율): **외국인 고객 24만 → 위임동의 → 사채보유 식별군 → 가심사 통과 → JB 본심사 통과**

| 시나리오 | 전환율 가정 | 연 이자수익(총, 차감 전) |
|---|---|---|
| 보수 | 3% | 약 28.5억 (자체추정) |
| 기준 | 7% | 약 66.5억 (자체추정) |
| 공격 | 10% | 약 95억 (자체추정) |
- ⚠️ **위 숫자는 총이자(gross)** — 여기서 **예상손실률(출국·회수불능, LGD)을 차감해야 순기여**. 손실률은 JB 외국인 신용대출 실데이터로 확정. → "보수 시나리오에서도 (+)"를 보이는 게 목표.
- 고객 가치: **1인 연 246만 절약**(개인조건 가정의 추정·범위 병기). → **고객 win + 은행 win 동시**.
- RORWA 언어로: "신규 모객비 0원, 휴면·기존 고객의 **위험조정 순기여**" (회장 4대축 정합).

## 신설 슬라이드 3 — 마중 P&L · 해자 · 시장 (사업 급소: 수익 주체·해자 실종)

- **시장(천장 들어올리기):** SOM 전북 24만 → SAM E-9 전국 약 32만+근로 체류자 → **TAM 한패스 동남아 송금 회랑 × AP2 표준**.
