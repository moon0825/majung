# 마중(Majung) — JB 실무진 "바로 꽂아 쓰겠다" 설득 분석

> 레포 실제 코드/문서를 직접 읽고 작성. 규제 사실은 출처 검증. "자체 추정"은 추정으로 표기. 코드 미수정(분석·권고).
> **핵심 진단:** 현재 데모는 "외부인이 보기 좋은 데모"로는 완성도가 높으나, JB 실무진이 "우리 시스템에 붙인다"고 느끼기엔 ① 모의(mock) 티가 곳곳에 남아 있고 ② 자기 부서의 KPI/통제지표 언어로 번역되어 있지 않다. **게이트/룰/송금경로(동결 구역)는 건드리지 않고, 표시·문서·연출 레이어에서 "JB 부서 언어"를 입히는 것**이 가장 효율적인 무브.

## 1) 페르소나 4종별 갭과 무브

### ① 외국인금융 사업담당
- **신경쓰는 것:** 기존 자산(휴면 20만+외국인 24만) 활성화·전환율, 점유율 72% 방어, 한패스·브라보 시너지, 규제사고 없는 KPI 달성.
- **갭:** 거시 수치(693억·66.5억·7%)가 전부 정적 "자체 추정"(demoData.js BIZ 하드코딩), 세션과 무연동. 퍼널이 행동과 연동 안 됨. 한패스 레일이 mock.
- **무브:** (low) 전환율 3/5/7/10% **민감도 표** 토글 → "7% 한 점 베팅" 인상 제거. (med) "JB 실데이터로 D+30 실측 가능" 슬라이드. (med) 퍼널을 세션 trace와 연동.

### ② 컴플라이언스/AML 담당
- **신경쓰는 것:** STR/CDD/CTR 적시·정확, 오탐 최소화, 룰 설명가능성·감사추적, 자동화가 사람 최종판단을 대체 안 함, 책임 소재.
- **갭:** AML 룰이 단순 가중합·임계값 임의값. structuring/just_below_threshold는 **필드만 있고 탐지 로직 없음**(항상 False). STR 큐가 적재만, **담당자 처리 워크플로 없음**. **CDD/CTR 개념 화면에 전무.** late_night 가산 0이 설명 없으면 오해 소지.
- **무브:** (low) STR 행에 **검토/보고/기각 버튼+담당자·시각 컬럼**. (low) **"보류 사유" 룰 트리거 설명 패널**(score 75=out_of_mandate 30+new 25+high 20) — flags 한국어 라벨만 풀면 됨. (med) **CDD/CTR 배지**를 게이트 트레이스에. (med) 발표 멘트 "가중치·임계값은 JB 룰북에 맞춰 파라미터 교체 가능"(AML_WEIGHTS 상수라 실제로 그러함). (high) structuring 실탐지 1개 — 룰 추가라 신중, 발표는 "로드맵".

### ③ 여신·심사 담당
- **신경쓰는 것:** 가심사가 본심사와 괴리되어 고객 기대 부풀리지 않을 것, DSR·다중채무·소득·비자 정합성, "사실상 승인처럼" 안 보일 것.
- **갭:** `prescreen_refi` DSR이 **하드코딩 0.32**(mcp_tools.py:267) — 즉시 짚을 최대 약점. 신용등급 `"가등급 B"` 고정(rules.py:205). 절약액이 **단리 1년치**인데 "연 절약액" 표기. jb_apr 13.59% 단일금리(등급 스프레드 없음).
- **무브:** (low) "가심사≠승인"+"본심사 시 금리·한도 변동 가능" 명문화. (med) **DSR 실값 계산 시연**(급여=소득, loans_external=부채 → prescreen_refi(dsr=...) 주입, 룰 무변경·호출부만). (med) **등급별 금리 테이블**(가A/B/C). (med) "마중은 고객을 JB 창구로 데워서(warm lead) 넘기는 깔때기" 프레이밍.

### ④ 디지털/IT·보안
- **신경쓰는 것:** 망분리, 외부 AI·MCP 데이터유출/프롬프트 인젝션, LLM 환각의 거래 영향, 개인신용정보, 가용성·감사로그·접근통제.
- **갭:** "LLM 송금경로 배제"는 코드로 증명된 강력 자산. 단 **"LLM에 어떤 데이터가 흘러가나"** 답이 데모에 없음. Claude Desktop 외부연동 컷이 IT/보안엔 **양날의 검**. 감사로그 무결성·보존기간 언급 없음. esign_hash 16자 절단(main.py:211). CORS `*`(main.py:38).
- **무브:** (low) **"LLM 전송=자연어 발화뿐, 계좌·실명·금액은 결정적 코드 내부에만" 데이터흐름도** 1장. (low) **금융위 2024.8 망분리 로드맵** 인용. (med) MCP 컷을 "행内 통제(도구별 권한·스코프 화이트리스트)"로 재연출. (med) 감사로그 체이닝 해시·보존정책 표기.

## 3) "그대로 붙이겠다" 보강 우선순위 8개 (레이어 / 동결구역)

1. **전금법 "접근매체 양도 아님" 방어 슬라이드** — 철회권+유효기간+범위→확정적 이전 아님, 자동이체 약정 동급. **[문서]** 무침범. *위법성 리스크 선제 차단. low·임팩트 최상.*
2. **"LLM에 흐르는 데이터=자연어뿐" 데이터흐름도 + 망분리 정렬 멘트** — 2024.8 로드맵 인용. **[문서/연출]** 무침범. *IT/보안 관문을 강점으로. low.*
3. **AML "보류 사유 설명 패널" + STR 처리 워크플로 버튼** — score 분해 한국어, 검토/보고/기각+담당자·시각. **[표시]** 룰 무변경(flags·score 표출). *설명가능성·워크플로 가시화. low~med.*
4. **DSR 실값 계산 시연** — 급여·loans_external로 산출해 prescreen_refi(dsr=) 주입(현재 0.32 하드코딩). **[코드]** **룰 무침범**(호출부만). *"가심사 아니다" 해소. med.*
5. **CDD·CTR 배지를 게이트 트레이스에** — 특금법 3종 완비 신호. **[표시/문서]** 무침범. *med.*
6. **거시 KPI 민감도 표(3/5/7/10%) + "JB 실데이터로 검증 가능"** — 단일추정 베팅 인상 제거. **[표시/문서]** 무침범. *med.*
7. **등급별 대환 금리 테이블 + 절약액 산식 정확화** — 가A/B/C 분기, "연 절약액=단리 1년치" 각주/다년 보정. **[코드/표시]** 룰 무침범(데이터·표시). *"숫자 책임"·금소법 정확고지. med.*
8. **감사로그 무결성(체이닝 해시)·보존정책 + 본인확인 수단 로드맵** — esign 모의 인정·"본선 시 JB 생체/공동인증서". **[표시/문서/코드]** 무침범. *전자서명 입증책임 대응. med.*

> **순서 논리:** 1~2 = 위법·보안 리스크 제거(책상에 올릴 전제조건). 3~5 = 각 부서 "이건 우리 업무다" 트리거. 6~8 = 신뢰도·정확도 마감. **동결 구역 침범 항목 없음** — 4·7의 코드 변경도 게이트 호출부/데이터 레이어에 국한.
