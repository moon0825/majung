## 0. 한 줄 전략 (심사위원이 듣고 싶은 언어)
> "마중은 JB가 2026년 명시적으로 밀고 있는 세 축 — **외국인 정착금융 × AX(AI 내재화) × RORWA(질적 여신성장)** — 에 동시 적중하며, '**LLM은 송금 실행경로에 없다**'는 제1원칙으로 은행이 가장 두려워하는 AI 환각·오송금·AML 리스크를 설계로 차단한, **내일 당장 생태계에 끼울 수 있는 완성형 서비스**다."

근거: 김기홍 회장 2026 "AX가 전 그룹에 뿌리내리는 원년" + 4대축(RORWA / 외국인·전략여신 / IB / AX). 챌린지 본질 = 청년인재 발굴 + 서비스 소싱, 본선 진출자 입사 우대.

## 2. 차별성 포지셔닝
경쟁사 전원(토스 46만·5개어 / 신한 SOL글로벌론 / 하나EZ 16개어 / KB 자동예약송금 / 우리WON글로벌 35만 / 한패스 / 센트비)은 **"언어·수수료·속도" 같은 칸**에서 경쟁. 마중의 3축 — ①위임형 자동실행 ②사채 대환 신용사다리 ③LLM 실행배제 — 은 **누구도 제품화하지 않은 빈칸.** KB·우리의 "자동/예약송금"은 고정 스케줄 단순 트리거(환율·한도·사기 무관하게 그냥 보냄) — 마중처럼 위임장 범위 내 **판정해 보류/실행을 결정**하지 않음.

**모방난이도:** ① 법적 위임 프레임+건별 모국어 통지+무조건 철회권(카피 시 불완전판매 리스크 떠안음) · ② 전북 72%·24만·심사엔진·라이선스 생태계 해자(신한 "3개월 급여" 요건으로 안 받는 신규자가 정확히 사채로 빠지는 구간 — 부산 사건 연 154%·9,120명·162억으로 실증) · ③ 규제친화 거버넌스+감사로그. **"기능은 베껴도 생태계는 못 베낀다."**

**슬로건 후보:** (A) "보내는 앱은 많다. 결정을 위임받는 에이전트는 마중뿐." (B) "LLM은 통역만, 돈은 결정적 코드가 — 은행이 안심하고 켜는 외국인 자동금융."

## 4. 위험·반박지점 Top + 30초 방어 (→ `본선_QnA_방어카드.md` 반영)
1. **(최대취약) 접근매체 대여금지(전금법§6③) 위반?** → "접근매체를 받지도 보관하지도 않음. **고객이 본인인증으로 사전등록한 오픈뱅킹/자동이체 정기송금 권한** 위에서 JB 결정적 코드가 실행. 대법 2011도16167은 '양도=확정적 이전'으로 좁게 봄, 우리는 이전·대여 자체가 없음."
2. **'연 246만 절약' 모국어 제시 = 부당권유?** → "가심사·회부까지만, 판매·승인은 JB. '246만'은 추정치로 모국어 명시+범위 병기. 적합성·청약철회·위법계약해지권 내장 — 권유 아닌 정보제공."
3. **외부 LLM 망분리·환각 책임?** → "제1원칙: LLM은 실행경로에 없음. 외부 LLM은 의도파싱·안내만, 판정·실행은 망 내부 결정적 3중 게이트. 금융위 2024.8 망분리 로드맵+2024.12 AI 지원방안 흐름과 정렬." ⚠️2026.1 시행세칙 최종발효 재확인
4. **자동송금이 STR 약화?** → "반대. C게이트가 전신송금 100만 CDD·1천만 CTR·구조화를 상시 적용, 의심시 자동보류+모국어통지+STR 회부. 보고주체는 전북은행, 마중은 위탁수행 — 책임 명확."
5. **(수치) 송금 25조/전무/NIM?** → 사전 정정으로 무력화(액션 4·5·6).
6. **(전환율) 7% 근거?** → "전체 아닌 한패스 송금데이터+전북 보유고객 중 사채 시그널 식별군 분모. 미국 개인대출 절반 이상이 부채통합 목적 — 의도풀 두텁고 신뢰 인프라 선점." ⚠️국내 사채→제도권 전환율 1차통계 미검증

**⚠️미검증 명시 의무:** 브라보 휴면 20만 / 입국전 사채 1,500~2,000만(부산 사건으로 대체 권고) / 송금 25조 본문(403) / 2026.1 시행세칙 발효 / 국내 대환 전환율.

## 5. 핵심 권고 (D-7)
1. **발표 첫 슬라이드 = JB 회장 키워드 그대로 인용**(외국인×AX×RORWA) → "전략정합+생태계 즉시적용" 각인.
2. **수치 정정 5건은 발표자료 동결 전 필수** — Q&A에서 잡히면 신뢰 전체가 흔들림.
3. **작동 데모 + 접근매체 비보유 다이어그램 + AP2 매핑** 3종이 기술완성도·사업화 루브릭 직결 최대 레버.
4. **제1원칙을 규제 4종(접근매체·AML·망분리·환각) 방어의 공통 축으로 반복** — 화려한 agentic이 아니라 "안전하게 통제된 금융실행"이 JB 보수성과 맞물린 가점.
